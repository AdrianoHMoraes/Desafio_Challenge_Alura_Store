Projeto de análise de dados de 4 lojas. 
foram analisados os dados de vendas de categorias de produtos, faturamento e avaliação dos clientes
o intuito foi de ajudar o dono das lojas na tomada de decisão sobre qual loja ele deve vender observando o que os dados de cada uma apresentam.
foram gerados tabelas onde é possível ver os dados extratificados
foram gerados gráficos visuais para melhor entendimento dos padrões
foi sugerido uma ação  a ser implementada para levantar capital para outros investimentos.
